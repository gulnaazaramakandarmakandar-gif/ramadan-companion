# 🌙 Ramadan Companion

A beautifully designed Ramadan productivity dashboard built using **Python + Streamlit**.

## ✨ Features

- 📊 Iftar & Sehri Countdown (API Integrated)
- 📅 Ramadan Calendar
- ✅ Daily Checklist
- 💧 Hydration Tracker
- ✨ Daily Reflection Quotes
- ⚙️ Custom City Settings

## 🛠 Tech Stack

- Python
- Streamlit
- Requests API
- Pandas
- Custom CSS Styling

## 🚀 How to Run Locally

```bash
pip install -r requirements.txt
streamlit run main.py
```
